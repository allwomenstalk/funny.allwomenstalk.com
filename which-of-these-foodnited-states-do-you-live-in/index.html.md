# Which of These Foodnited States do You Live in?

Artist Chris Durso and his 8-year-old son are working on a U.S. map I can totally relate to, and so can any other foodie, American or other. They're making funny, clever wordplay puns on the names (and fave foods?) of each state... have they done your home state yet? I don't see mine yet (The Mitten State), but I'll be looking out for it.
